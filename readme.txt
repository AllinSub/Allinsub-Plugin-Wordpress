== Description ==
Easy Premium helps content creator for adding subscription model to their blog.

Allinsub is free to install (just fees on transactions).

For your users it\'s an easy way to aggregate their subscriptions for their loved websites.

== Installation ==
After adding the plugin in your wordpress directory, add your EditorId. You can find it on http://allinsub.com/myaccount.
